# ☀️ SunTracker (C + WebAssembly)

A high-performance solar elevation calculator built in C and compiled to WebAssembly, with a JavaScript frontend for visualization.

* * *

## 🚀 Features

* Calculate real-time solar elevation
* Generate a full-day elevation graph
* High-performance math using C + WebAssembly
* Interactive visualization using Chart.js
* Clean, responsive UI

* * *

## 🧠 Tech Stack

* C – solar calculations
* WebAssembly (Emscripten) – performance layer
* JavaScript (ES Modules) – frontend logic
* Chart.js – data visualization
* HTML/CSS – UI

* * *

## ⚙️ Build Instructions

### 1. Install Emscripten

git clone <https://github.com/emscripten-core/emsdk.gitcd> emsdk./emsdk install latest./emsdk activate latest./emsdk_env.sh # Windows: .ps1

* * *

### 2. Compile C → WebAssembly

emcc main.c -O3 -s WASM=1 -s EXPORTED_FUNCTIONS='["_get_solar_elevation"]' -s EXPORTED_RUNTIME_METHODS='["cwrap"]' -o main.js

* * *

### 3. Run locally

npx serve

or

python -m http.server

* * *

## 📊 Usage

1. Enter:
  
* Latitude
* Longitude
* Altitude

2. Click:
  
* Get Current Elevation → shows current solar elevation
* Gen Chart → displays full-day solar elevation graph

* * *

## 🔌 WebAssembly API

double get_solar_elevation(...);

JavaScript:

Module.cwrap("get_solar_elevation", "number", [...]);
